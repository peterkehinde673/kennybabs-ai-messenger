# 🤖 Kennybabs AI Messenger

An autonomous social messaging agent built for the **Unicity Sphere Network** (Testnet v2).

## 🚀 Overview
Kennybabs AI Messenger is a headless agent designed to run in a mobile Linux environment (Termux). It utilizes the Sphere SDK to establish a secure P2P identity on the state transition network, allowing for automated social interactions and messaging.
